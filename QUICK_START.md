# CamHome - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Download
1. Go to: https://github.com/bhartiarun05/camera/releases
2. Download the latest `CamHome.exe` or `CamHome-Setup.exe`
3. Save to your Desktop or any folder

### Step 2: Run
- **Portable Version**: Double-click `CamHome.exe` - No installation needed!
- **Installer Version**: Double-click `CamHome-Setup.exe` and follow the installer

### Step 3: Add Cameras
1. Click "Add Camera" button
2. Choose discovery method:
   - **Scan QR Code** - Point at camera QR code
   - **Add by IP Address** - Enter IP address
   - **Search Network** - Auto-scan LAN
   - **Manual Entry** - Enter stream URL
3. Click "Add Camera"

## 📱 Camera Discovery Methods

### QR Code Scanner
✅ Real-time scanning
✅ Auto configuration
✅ One-click setup

### IP Address
✅ Manual IP entry
✅ Auto port detection (80, 8080, 554, 8000, 8081, 5000, 9000)
✅ Protocol detection

### Network Search (LAN)
✅ Automatic device discovery
✅ Local network scanning
✅ Port detection
✅ One-click add

### Manual Entry
✅ RTSP streams
✅ HTTP streams
✅ MJPEG streams
✅ Custom credentials

## 🎯 Stream URL Examples

```
RTSP:   rtsp://username:password@192.168.1.100:554/stream
HTTP:   http://192.168.1.100:8080/video.mjpeg
MJPEG:  http://192.168.1.100/mjpeg.cgi
```

## 💻 System Requirements

- Windows 11
- 4GB RAM (8GB recommended)
- 500MB disk space
- Active internet connection

## 🆘 Troubleshooting

**Camera not found?**
- Verify IP address
- Check network connectivity
- Ensure firewall allows connections
- Try manual port entry

**QR scanner not working?**
- Enable camera permissions
- Ensure good lighting
- Hold steady and closer to QR code

**App won't start?**
- Check Windows 11 compatibility
- Run as Administrator
- Disable antivirus temporarily
